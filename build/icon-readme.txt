Иконки для сборки (можно сгенерировать из одной картинки):

1) Положите вашу иконку в эту папку как icon-source.png (или icon-source.jpg, или icon.png).
2) В корне проекта выполните:  npm run convert-icon

Скрипт создаст:
  - icon.png (256×256) — иконка приложения для electron-builder
  - icon.ico (256, 48, 32, 16) — для Windows exe/установщика
  - tray.png (32×32) — иконка в системном трее

Если tray.png отсутствует, приложение запустится без иконки в трее.
