# Qbit Remote GUI

Десктопное приложение для удалённого управления qBittorrent через официальный Web UI. Работает как **Transmission Remote GUI**: отдельное окно с тем же интерфейсом и функционалом, что и веб-интерфейс в браузере.

Используется **официальный Web UI** qBittorrent: приложение открывает в своём окне ваш запущенный qBittorrent Web UI (локально или по сети). Никакой копии интерфейса — подключаетесь к тому же UI, что и в браузере. В заголовке окна отображается **Qbit Remote GUI (qBittorrent vX.X.X)**.

## Требования

- **Node.js** 18+
- Запущенный **qBittorrent** с включённым Web UI (Настройки → Web UI).

По умолчанию Web UI слушает порт **8080** (`http://127.0.0.1:8080`).

## Установка и запуск

```bash
npm install
npm start
```

При первом запуске откроется окно «Подключение» — введите адрес Web UI (например `http://127.0.0.1:8080` или `http://IP_СЕРВЕРА:ПОРТ`) и нажмите «Подключиться». Адрес сохраняется и подставляется при следующих запусках.

## Сборка установщика (Windows)

```bash
npm run dist
```

Готовый установщик: `dist/Qbit Remote GUI Setup 1.0.0.exe` (и папка `dist/win-unpacked` для запуска без установки).

## Иконки перед сборкой

Чтобы приложение и установщик использовали вашу иконку:

1. Положите исходную иконку в папку `build/` как **icon-source.png** (или icon-source.jpg, или icon.png).
2. Выполните:
   ```bash
   npm run convert-icon
   ```
3. В `build/` появятся **icon.png** (256×256), **icon.ico** (для exe и установщика), **tray.png** (32×32 для трея).

Подробнее — в `build/icon-readme.txt`.

## Возможности

- Окно приложения с полным Web UI qBittorrent (торренты, настройки, логи и т.д.).
- **Добавление торрента** (двойной клик по .torrent, меню **Файл → Добавить торрент…** или Ctrl+O):
  - папка загрузки, «Сохранить как», чекбокс «Запустить торрент»;
  - **категория** — выбор из списка категорий с сервера; по умолчанию подставляется последняя папка пути (например путь `D:/Anime/Ongoing` → категория `Ongoing`);
  - выбор файлов в раздаче, OK/Отмена.
- После установки можно выбрать «Открывать с помощью» для файлов .torrent.
- Окно подключения: смена адреса (локальный/удалённый сервер), логин и пароль Web UI.
- Сохранение последнего адреса, учётных данных и последних папок загрузки между запусками.
- Меню: «Подключение…», «Отключиться от сервера», обновление, полный экран, «О Qbit Remote GUI».
- Иконка в трее (если есть `build/tray.png`).
- Один экземпляр приложения (повторный запуск или открытие .torrent открывает уже запущенное окно).

## Скрипты

| Команда           | Описание |
|-------------------|----------|
| `npm start`       | Запуск приложения |
| `npm run dist`    | Сборка установщика Windows (NSIS) в `dist/` |
| `npm run pack`    | Сборка без установщика (папка `dist/win-unpacked`) |
| `npm run convert-icon` | Генерация icon.png, icon.ico и tray.png из иконки в `build/` |
